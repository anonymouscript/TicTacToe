This Repository Contains TicTacToe.py

When TicTacToe.py is run, it will find the scripts in the players folder and pit them against each other calling their move method. 

Scoring:

Losing: 0 points
Tie/Win: 1 points (Tic Tac Toe is a Zero Sum game)

If your program throws an error or exception, it will be distqualified.
